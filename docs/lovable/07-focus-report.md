# Step 7: Personal Focus Report Page

> Paste this into Lovable. **Do not modify existing pages.** Reuse `mockSession` data.

---

Build the Focus Report page at route `/session/:id/report`. This is a shareable one-page summary — think "Spotify Wrapped" but for study focus. Designed to look good as a screenshot.

## Layout

This page should feel like a **card/poster** — centered, fixed-width (max 600px), visually bold.

### Top: Header
- Session date: "Tuesday, March 28, 2026"
- Session duration: "30 minutes"
- Small Pawsed logo/wordmark

### Hero: Overall Score
- A large circular gauge/ring (donut shape) centered on the page
- Shows "72.5%" in the center, large and bold
- Ring color: green for the filled portion, muted/dark for the empty portion
- Label below: "Focus Score"
- Build this with an SVG circle or use Recharts RadialBarChart

### Section: Top 3 Distraction Patterns
- Three cards in a row (or stacked on small screens)
- Each card shows:
  - An icon (eye, mouth, arrow)
  - The distraction type: "Looked Away", "Yawn", "Eyes Closed"
  - The count: "7 times", "3 times", "2 times"
  - Sorted by frequency (most common first)

### Section: Session Highlights
- Two side-by-side stat blocks:
  - "Longest Focus Streak: **11 min**" with a flame icon
  - "Biggest Danger Zone: **12:00–18:00**" with a warning icon

### Section: Comparison (if multiple sessions exist)
- For now, show mock comparison:
  - "vs. Your Average: **+5.2%**" with an upward arrow in green
- If this is the only session, show: "Complete more sessions to see trends"

### Bottom: Share Actions
- "Share Report" button → copies a link to clipboard (show toast "Link copied!")
- "Download as Image" button → for now, just show the button (functionality can be added later)
- Small footer: "Generated by Pawsed"

## Style
- **Bold and visual** — this is meant to be screenshotted and shared
- Use large font sizes for numbers (3rem+)
- Background: slightly different from the rest of the app — use a subtle gradient from slate-900 to slate-800
- Cards should have slightly more visual flair than other pages — subtle glow/shadow effects are OK here
- The circular gauge should be the visual centerpiece
- Use the session sub-navigation tabs (Report tab active)
